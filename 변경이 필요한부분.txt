현재 프로젝트에 ortools를 사용해서 근무표를 생성하는 방식이 아닌

openai api 에 근무표를 생성요청을 하고 그 응답을 
json을 생성하여 응답하는 방식으로 구현하면 어떨까  
- 어떤 근무인원이든 어떤 최대근무시간이든 어떤 근무형태이든 openai가  형평성에 맞게 생성하게 해주는 (좀더 현실적인)

openai 에 현재 실제 근무인원과 제약을 던지고 
근무표 데이터를 json으로 받아서 그대로 cpp 서버로 기존 프로토콜 지켜서 던지는 방식이며
생성시도 하였으나 해가 없음이 나올시에는 openai가 대안의 근무표를 생성해준다.
이 버전을 server_ai_gen.py로 생성 



그럼 server_ai_gen.py와 shift_server_optimized.py 서버 각각의 함수와클래스별 역할에 대해 상세히 기술하는
기술명세서 작성해줘


  ChangeShift 프로토콜 JSON 형식:
  {
    "Protocol": "ChangeShift",
    "staff_id": "1",
    "date_": "2025-08-10",
    "shift_from": "D",
    "shift_to": "E"
  }

  주요 변경사항:

  1. 함수 시그니처 변경: updateAndExecuteShiftScheduler 함수가 이제 4개의 개별 매개변수를 받습니다
    - staff_id: 직원 ID
    - date_: 날짜 (YYYY-MM-DD 형식)
    - shift_from: 기존 근무 시간대
    - shift_to: 변경할 근무 시간대
  2. JSON 처리 개선: ChangeShift 프로토콜에서 4개의 필드를 직접 추출하여 처리
  3. 에러 처리: 필수 정보가 누락된 경우 적절한 에러 메시지 반환
  4. 자연어 파싱 제거: 더 이상 자연어를 파싱할 필요가 없고 구조화된 데이터를 직접 사용

  이제 C# 클라이언트에서 구조화된 JSON 데이터로 근무교대 요청을 보낼 수 있으며, 서버는 이를 처리하여 적절한 응답을 반환합니다.
